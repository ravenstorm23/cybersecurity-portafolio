# cybersecurity-portfolio

Repositorio personal de ciberseguridad enfocado en el análisis técnico de documentos maliciosos, vulnerabilidades y técnicas de ejecución.

Incluye laboratorios prácticos con análisis estático y dinámico, identificación de procesos, mapeo a MITRE ATT&CK, reglas de detección (SIEM) y documentación basada en CVEs reales, desarrollados en entornos controlados con fines educativos y defensivos.
